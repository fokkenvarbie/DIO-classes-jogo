<h1>Classificador de Ataques de Heróis</h1>

Um programa em JavaScript que exibe seus ataques de acordo com a classe de cada campeão. O sistema permite definir nome e classe do herói e exibe no console a ação realizada.

- **Funcionalidades:**
  - **Definição de heróis:** Permite criar heróis com nome e classe (Mago, Guerreiro, Monge ou Ninja).
  - **Método de ataque:** Cada herói possui um método atacar() que determina o ataque automaticamente com base na sua classe.
  - **Classificação por classe:** Cada tipo de herói possui um ataque característico: magia, espada, artes marciais ou shuriken.
  - **Estruturas de decisão:** Utiliza condicionais (switch) para definir qual ataque será exibido.
  - **Reutilização e escalabilidade:** Permite adicionar qualquer quantidade de heróis e executar seus ataques de forma organizada.
  - **Saída clara:** Exibe mensagens no console como "Ahri (Mago) atacou usando Magia", mostrando nome, classe e ataque do herói de forma clara e estruturada.
